# SQL Server Performance Tuning with AdventureWorks2017

This project demonstrates fundamental SQL Server performance tuning techniques
using the AdventureWorks2017 sample database.

## Key Concepts Covered
- Indexing (Non-Clustered & Composite)
- Execution Plan Analysis
- Query Statistics (IO & TIME)
- Query Optimization Best Practices
- CTE-based optimization

## Tools Used
- SQL Server
- SQL Server Management Studio (SSMS)
- AdventureWorks2017

## Results
- Improved query performance using indexes
- Reduced logical reads and execution time
- Clear before/after execution plan comparisons

## Why This Matters
Understanding performance tuning fundamentals is critical for building scalable,
efficient data systems in production environments.
